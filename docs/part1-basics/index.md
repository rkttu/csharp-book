# 기초편

C# 프로그래밍의 기초를 탄탄히 다지는 과정입니다.

## 학습 개요

기초편에서는 C# 개발 환경 설정부터 시작하여 프로그래밍의 핵심 개념들을 단계적으로 학습합니다. 변수, 제어문, 데이터 구조, 함수 등 모든 프로그래밍 언어의 기본이 되는 개념들을 File-based App 방식으로 쉽게 배울 수 있습니다.

## 구성

### 제1부: 시작하기

Visual Studio Code와 .NET 10 환경을 설정하고 첫 번째 프로그램을 만듭니다.

- **1장. 개발 환경 구축** - VS Code와 .NET SDK 설치
- **2장. C# 기초 문법** - 변수, 데이터 타입, 연산자
- **3장. 제어문** - if, switch, for, while 등

### 제2부: 데이터 구조와 컬렉션

다양한 형태의 데이터를 효율적으로 저장하고 관리하는 방법을 배웁니다.

- **4장. 배열** - 1차원, 다차원 배열
- **5장. 컬렉션** - List, Dictionary, HashSet 등
- **6장. 문자열 처리** - 문자열 조작과 StringBuilder

### 제3부: 함수와 메서드

코드의 재사용성과 모듈화를 위한 메서드 작성법을 익힙니다.

- **7장. 메서드 기초** - 메서드 정의와 호출
- **8장. 고급 메서드 활용** - 람다 식, 확장 메서드, 재귀

## 학습 목표

기초편을 마치면 다음을 할 수 있습니다:

- ✅ C# 개발 환경을 스스로 구축하고 관리할 수 있습니다
- ✅ 변수, 연산자, 제어문을 자유롭게 사용할 수 있습니다
- ✅ 배열과 컬렉션으로 데이터를 효율적으로 관리할 수 있습니다
- ✅ 문자열을 능숙하게 처리할 수 있습니다
- ✅ 메서드를 작성하여 코드를 모듈화할 수 있습니다
- ✅ 간단한 콘솔 애플리케이션을 처음부터 끝까지 만들 수 있습니다

## 학습 시간

기초편은 약 **15-20시간** 정도의 학습 시간이 필요합니다. 각 예제를 직접 입력하고 실행해보면서 학습하시길 권장합니다.
