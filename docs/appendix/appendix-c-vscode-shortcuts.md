# 부록 C. Visual Studio Code 단축키

VS Code를 효율적으로 사용하기 위한 필수 단축키를 정리했습니다. Windows/Linux와 macOS의 단축키를 함께 표기합니다.

**표기법:**

- `Ctrl` (Windows/Linux) = `Cmd` (macOS)
- `Alt` (Windows/Linux) = `Option` (macOS)

## C.1 기본 편집

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 저장 | `Ctrl+S` | `Cmd+S` |
| 모두 저장 | `Ctrl+K S` | `Cmd+K S` |
| 실행 취소 | `Ctrl+Z` | `Cmd+Z` |
| 다시 실행 | `Ctrl+Y` | `Cmd+Shift+Z` |
| 잘라내기 | `Ctrl+X` | `Cmd+X` |
| 복사 | `Ctrl+C` | `Cmd+C` |
| 붙여넣기 | `Ctrl+V` | `Cmd+V` |
| 줄 복사 | `Shift+Alt+↓/↑` | `Shift+Option+↓/↑` |
| 줄 이동 | `Alt+↓/↑` | `Option+↓/↑` |
| 줄 삭제 | `Ctrl+Shift+K` | `Cmd+Shift+K` |

## C.2 코드 탐색

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 파일 열기 | `Ctrl+P` | `Cmd+P` |
| 기호로 이동 | `Ctrl+Shift+O` | `Cmd+Shift+O` |
| 정의로 이동 | `F12` | `F12` |
| 참조 찾기 | `Shift+F12` | `Shift+F12` |
| 뒤로/앞으로 | `Alt+←/→` | `Ctrl+-/Ctrl+Shift+-` |
| 줄로 이동 | `Ctrl+G` | `Cmd+G` |

## C.3 디버깅

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 디버깅 시작 | `F5` | `F5` |
| 디버깅 중지 | `Shift+F5` | `Shift+F5` |
| 다시 시작 | `Ctrl+Shift+F5` | `Cmd+Shift+F5` |
| 계속 | `F5` | `F5` |
| Step Over | `F10` | `F10` |
| Step Into | `F11` | `F11` |
| Step Out | `Shift+F11` | `Shift+F11` |
| 중단점 토글 | `F9` | `F9` |

## C.4 터미널

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 터미널 열기 | ``Ctrl+` `` | ``Cmd+` `` |
| 새 터미널 | ``Ctrl+Shift+` `` | ``Cmd+Shift+` `` |
| 터미널 간 이동 | `Ctrl+PgUp/PgDn` | `Cmd+PgUp/PgDn` |
| 터미널 닫기 | `Ctrl+W` | `Cmd+W` |

## C.5 검색 및 바꾸기

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 찾기 | `Ctrl+F` | `Cmd+F` |
| 바꾸기 | `Ctrl+H` | `Cmd+H` |
| 파일에서 찾기 | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| 다음 찾기 | `F3` 또는 `Enter` | `F3` 또는 `Enter` |
| 이전 찾기 | `Shift+F3` | `Shift+F3` |

## C.6 유용한 기능

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 명령 팔레트 | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| 빠른 열기 | `Ctrl+P` | `Cmd+P` |
| 설정 열기 | `Ctrl+,` | `Cmd+,` |
| 사이드바 토글 | `Ctrl+B` | `Cmd+B` |
| 다중 커서 | `Ctrl+Alt+↓/↑` | `Cmd+Option+↓/↑` |
| 선택 확장 | `Shift+Alt+→` | `Shift+Option+→` |
| 주석 토글 | `Ctrl+/` | `Cmd+/` |
| 블록 주석 | `Shift+Alt+A` | `Shift+Option+A` |

## C.7 C# 전용 단축키

| 동작 | Windows/Linux | macOS |
|------|---------------|-------|
| 코드 정렬 | `Shift+Alt+F` | `Shift+Option+F` |
| Quick Fix | `Ctrl+.` | `Cmd+.` |
| 이름 바꾸기 | `F2` | `F2` |
| Peek 정의 | `Alt+F12` | `Option+F12` |

---

**팁:** 명령 팔레트(`Ctrl+Shift+P`)에서 "Keyboard Shortcuts"를 검색하여 모든 단축키를 보거나 커스터마이징할 수 있습니다.
